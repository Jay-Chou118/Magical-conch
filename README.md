# Magical-conch

Near-field communication based on ultrasonic waves 基于超声波的近场通信
音频频率传输工具，支持跨平台音频信号处理与传输

## 支持平台
- macOS (已测试)
- Linux (Ubuntu/Debian)
- Windows
- Android (实验性)

## 依赖项

### macOS
```bash
brew install portaudio sdl2
```

### Linux
```bash
sudo apt install libportaudio2 libsdl2-dev
```

### Windows
```bash
choco install portaudio sdl2
```
